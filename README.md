
# Nombre del proyecto
Un estudio de viabilidad y un marco metodológico para la aplicación de modelos de machine learning en el nexo WEF en Galápagos: Oportunidades y desafíos ante la escasez de datos

---

# Características principales

-Aplicación de XGBoost se aplican las librerías xgboost, XGBRegressor desde xgboost.sklearn, MinMaxScale de sklearn.preprocessing para realizar la predicción de la producción agricola 

# Tecnologías utilizadas

- Python 3.x  
- Pandas, NumPy, Matplotlib  
-Google colab
